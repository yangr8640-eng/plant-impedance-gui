BMI 硬件 - Windows 版电阻曲线 GUI

使用方法：
1. 先安装 Python 3，并在安装时勾选 Add python.exe to PATH。
2. 双击“启动电阻曲线GUI_Windows.bat”。
3. 启动器会自动检查并安装 pyserial 串口依赖。
4. 选择硬件对应的 COM3、COM4 等串口。
5. 选择和 SSCOM 一致的波特率。
6. 在实验变量区的“测试时长(s)”中填写本次实验时长，例如 300 表示 5 分钟，1800 表示 30 分钟。
7. 填写其他实验变量：实验时间、温度、湿度、电极位置、土壤状态和植物编号。
8. 如果曲线被自动放得太大，可在“Y格差(Ω)”中填写每个 y 轴单元格的电阻差，例如 10 表示每格 10Ω；留空表示自动。
9. 可使用“放大”“缩小”“重置缩放”查看曲线细节；在图像上拖移可平移，触控板滚动或鼠标滚轮可缩放；鼠标移到曲线点附近会显示该点的时间和电阻。
10. 点击“连接”。
11. 实验结束后可点击“导出原始CSV”，保存芯片发来的原始文本行；也可点击“导出PDF”，保存包含实验变量、平均值、极值和曲线图的实验报告。

时间轴说明：
- x 轴会从 0 秒显示到你填写的测试时长。
- 计时从第一条有效电阻数据开始。
- 到达设定测试时长后，GUI 会自动停止读取串口或停止模拟数据，曲线不会继续追加。
- GUI 中的图像 y 轴、顶部统计值和悬浮提示统一使用 kΩ。

注意：
- 如果 SSCOM 正在连接同一个串口，请先在 SSCOM 中关闭串口。
- 如果 pyserial 自动安装失败，可以手动运行：
  python -m pip install pyserial
- 曲线流畅度主要取决于硬件发送频率。建议硬件每 10-50 ms 发送一次，例如：
  R=1023.5 ohm
- “导出原始CSV”会保存芯片发来的原始文本行，并附带接收时间和解析出的电阻值。
- PDF 不再导出每个时间点的原始数据，避免报告过长。
- 如果芯片一行里同时输出 i、Rs、Rr，例如 “i=608 Rs=142735 Rr=369”，GUI 会优先把 Rs 当作电阻值绘图。
